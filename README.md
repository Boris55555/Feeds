# Feeds

A pure black and white RSS/Atom feed reader designed for E-ink devices.

## Features
- Pure black and white UI (no grayscale, no images)
- RSS/Atom feed support with discovery from website URLs
- Full article extraction using Readability4J
- Built-in translation support (gtx)
- Tag-based organization
- Offline archive support
- Export/Import subscriptions
